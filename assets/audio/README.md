# Carpeta de Audio

Coloca el archivo de música de fondo aquí con el nombre: `background_music.mp3`

El archivo debe ser:
- Formato: MP3 o WAV
- Nombre exacto: `background_music.mp3`
- Duración: Preferiblemente una música que se pueda repetir (2-5 minutos)

El archivo se reproducirá automáticamente en el menú con estas características:
- Volumen: 30% (de fondo)
- Reproducción: En bucle infinito
- Botón de silencio: Esquina superior derecha del menú
